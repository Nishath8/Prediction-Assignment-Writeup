## Overview
This repository contains the prediction assignment for the Coursera Practical Machine Learning course. The goal of the project is to predict the manner in which a barbell lift was performed using accelerometer data from wearable sensors.

## Files
- `Prediction_Assignment_Writeup.Rmd` – R Markdown analysis
- `Prediction_Assignment_Writeup.html` – Compiled HTML report

## Model
A Random Forest classifier was trained after preprocessing the data and removing irrelevant features. The model achieved approximately 99.5% accuracy on the validation dataset.
